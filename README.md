# Node-Auth
Node-Auth is a simple user authentication system API built in Node.js.

## Features
- User registration
- User login
- Email verification
- Password reset

## Requirements
- Node.js
- MongoDB

Run the following command to install all dependencies

```
npm install
```

## Usage
Go to the root directory and run the following command to start the server

```
node src/server
```
